函式與參數

一、什麼是函式

  • 一段可以重複使用的程式邏輯。
  • 可以封裝程式區塊，接收外部資料 (參數)，並傳出結果 (回傳值)。
  • 帶有參數的函式，此參數可以當作函式裡的臨時變數使用。

二、範例練習

1、定義 + 呼叫函式 

                            function sayHello() {
                                console.log('您好！');
                            }

                            sayHello(); //呼叫 sayHello 函式
                            sayHello(); //可重複呼叫函式
2、呼叫帶有「參數」的函式 

                            // 括弧裡面的 name 是參數
                            function greet(name) {
                                console.log(`您好，${name}。`);
                            }

                            greet('小明');
                            greet('小華');
3、呼叫帶有「回傳值」的函式 (一定要有 return) 

                            function add(x, y) {
                                return x + y;
                            }
                            
                            let addResult = add(5, 2);
                            console.log(addResult);

需要 return / 不需要 return 使用情況

情況 是否需要用 return 範例
計算總和、平均、面積等 ✔是 return x + y、return score >= 60
判斷條件 (是 / 否) ✔是 return isVIP && score > 90
將結果儲存在變數裡 ✔是 let a = getValue()
只是顯示內容 (畫面 或 console) ❌否 console.log(...)

.toStrin() / .toFixed() / parseInt() / parseFloat()

名稱 功能 回傳型別 範例 結果 補充說明
.toString() 將數字轉為字串 string (3.14159).toString() "3.14159" 顯示格式為"數字"的字串
.toFixed(n) 取小數點後 n 位數 string (3.14159).toFixed(2) "3.14" 顯示格式為"字串"的數字。
parseInt(str) 將字串轉為整數 number parseInt("33.57") 33 小數點後會被忽略。
parseFloat(str) 將字串轉為浮點數 number parseFloat("33.57") 33.57 可處理小數點，通常搭配.toFixed()使用。
number(str) 將字串轉為整數或小數 number Number("33.57") 33.57 和 parseFloat() 類似，但更嚴格。

使用建議

場景 建議用法
呈現格式 (畫面顯示) 用.toFixed()回傳字串即可。
邏輯判斷 / 數學運算 用 parseFloat((...).toFixed()) 轉回數字
處理整數輸入 用 parseInt()
更嚴格將字串轉換數字 用 Number() (如果回傳 NaN 代表錯誤)

BMI 分級表

BMI 值 狀態
BMI < 18.5 過輕
18.5 <= BMI < 24 正常
24 <= BMI < 27 過重
27 <= BMI < 30 輕度肥胖
30 <= BMI < 35 中度肥胖
BMI >= 35 重度肥胖

                        let spanWeight = document.getElementById('spanWeight');
                        let spanHeight = document.getElementById('spanHeight');
                        let spanBMI = document.getElementById('spanBMI');
                        let spanBMIShow = document.getElementById('spanBMIShow');

                        function fnBMI(weight, height) {
                            return parseFloat((weight / (height ** 2)).toFixed(1));
                        }

                        let weight = 62;
                        let height = 165;
                        let bmi = fnBMI(weight, height / 100);

                        spanWeight.textContent =`${weight} 公斤`;
                        spanHeight.textContent = `${height} 公分`;
                        spanBMI.textContent = `${bmi}`;

                        if (bmi < 18.5) {
                            spanBMIShow.textContent = '過輕';
                            spanBMIShow.className = 'text-secontary';
                        } else if (bmi < 24) {
                            spanBMIShow.textContent = '正常';
                            spanBMIShow.className = 'text-success';
                        } else if (bmi < 27) {
                            spanBMIShow.textContent = '過重';
                            spanBMIShow.className = 'text-orange';
                        } else if (bmi < 30) {
                            spanBMIShow.textContent = '輕度肥胖';
                            spanBMIShow.className = 'text-warning';
                        } else if (bmi < 35) {
                            spanBMIShow.textContent = '中度肥胖';
