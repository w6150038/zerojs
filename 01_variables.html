變數與資料型別

一、什麼是變數

是用來「儲存資料」的容器，可以反覆使用與更新。

宣告方式： 

                            let name = '小明';
                            const age = 18;
  • let：可以重新指定內容 (最常用)。
  • const：不可以改變 (常用來保存設定值)。

二、資料型別

型別 範例 說明
字串 (String) "Hello"、'您好' 用雙引號或單引號包起來。
數字 (Number) 10、3.14 不用引號，直接寫數值。
布林 (Boolean) true、false 只有兩種值，用於判斷邏輯。
空值 (Null) null 明確表示「沒有值」。
未定義 (Undefined) undefined 還沒指定的變數值。
陣列 (Array) [1, 2, 3] 一組資料清單。
物件 (Object) {name: "小明", age: 18} 對應資料集合。

三、範例練習


                            // 宣告一個變數：使用者姓名。
                            let userName = '小明';

                            // 宣告一個常數：使用者年齡。
                            const userAge = 15;

                            // 宣告是否為數字：是。
                            let isNumber = true;
                            // 印出所有變數。

                            console.log(userName);
                            console.log(userAge);
                            console.log(isNumber);

四、練習題

  1. 宣告一個變數 schoolName，內容為 '57國小'。
  2. 宣告一個常數 pi，內容為 3.14159。
  3. 用 console.log 印出內容。

                            let schoolName = '57國小';
                            const pi = 3.14159;

                            console.log(schoolName);
                            console.log(pi);
