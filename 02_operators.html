條件判斷與邏輯運算子

一、if / else

根據條件執行不同的程式邏輯。

基本語法： 

                            if (條件) {
                                條件成立時執行這段程式
                            } else {
                                條件不成立時執行這段程式
                            }
範例練習： 

                            let age1 = 18;

                            if (age1 >= 18) {
                                console.log('已成年！');
                            } else {
                                console.log('未成年！');
                            }

二、比較運算子

比較符號 說明 範例 (回傳布林值)
== 相等 (值相同) 5 == "5"，回傳 true。
=== 嚴格相等 (值和型別必須相同) 5 === "5"，回傳 false，因為左邊是數字，右邊是字串。
!= 不等於 3 != 5，回傳 true。
>, <, >=, <= 大小比較 10 > 5，回傳 true。

三、邏輯運算子

邏輯符號 名稱 用法 範例
&& AND 兩邊都要為 true 才執行。 age >= 18 && isNumber
|| OR 兩邊只要有一邊為 true 就執行。 age >= 18 || isNumber
! NOT 取反值。 !isLogin (如果沒登入)
範例練習： 

                            let username = "admin";
                            let password = "1234";
                            
                            if (username === "admin" && password === "1234") {
                                console.log("登入成功！");
                            } else {
                                console.log("登入失敗！");
                            }

五、多層判斷 else if

多層判斷語法： 

                            if (條件 1) {
                                條件 1 成立時執行這段程式
                            } else if (條件 2) {
                                條件 2 成立時執行這段程式
                            } else {
                                以上條件都不成立時執行這段程式
                            }
範例練習 (分數等級判斷)：
分數結果：

                            let score2 = 78;
                            let score2SpanResult = document.getElementById('score2SpanResult');
                            let score2DivResult = document.getElementById('score2DivResult');
                            
                            score2SpanResult.textContent = `${score2} 分`;
                            
                            if (score2 >= 90) {
                                score2DivResult.textContent = "超級優秀！";
                                score2DivResult.className = "alert alert-success";
                            } else if (score2 >= 60) {
                                score2DivResult.textContent = "恭喜及格！";
                                score2DivResult.className = "alert alert-primary";
                            } else {
                                score2DivResult.textContent = "不及格，需要加油！";
                                score2DivResult.className = "alert alert-danger";
                            }
