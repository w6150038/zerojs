第 1 階段：JavaScript 基礎 + Bootstrap 整合

課程階段 學習主題 課程說明
01 變數與資料型別 了解 JS 裡各種資料型別與宣告方式
02 運算子與條件判斷 比較運算子、邏輯運算子與 if/else 判斷邏輯
03 函式與參數 拆分邏輯、參數與回傳值
04 陣列與物件基礎 管理清單與資料組合 (例如相簿 / 表單欄位)
05 DOM 操作 (整合 Bootstrap) 用 JavaScript 控制畫面元素 (例如顯示 / 隱藏 / 修改樣式)
06 事件處理與互動 綁定 click / submit 等事件，處理互動邏輯
07 迴圈與反覆操作 使用 for / while / forEach 等處理清單資料與 DOM 輸出

課程階段 學習主題 課程說明
08 陣列方法 (push, splice) 新增、刪除、查找、排序清單內容，常用於購物車、列表管理等
09 巢狀資料與物件陣列 陣列包物件、物件包陣列，管理複雜結構 (如相簿清單、學生資料)
10 表單資料處理與驗證 使用者輸入、錯誤檢查、格式驗證、預防表單誤送出
11 計時器與時間控制 setTimeout、setInterval，應用在倒數、輪播等功能
12 錯誤處理與防呆 try/catch、檢查 undefined / null、提高程式穩定度

第 3 階段：Bootstrap + JavaScript 實務整合

課程階段 學習主題 課程說明
13 Bootstrap + JS 表單驗證整合 使用 Bootstrap 樣式與 JS 驗證配合 (登入 / 報名表單)
